# 📈 Marketing Predict: On-site Behavioral Analysis
## Predicting Real-Time Customer Intent

### 📌 Project Vision
Understanding customer intent during active sessions is key to personalization. This project explores predictive modeling based on on-site behavioral logs to forecast conversion and engagement.

### 🛠️ Analytical Focus
- **Session Analysis:** Feature engineering from dwell time, click sequences, and cart additions.
- **Conversion Modeling:** Using classification algorithms (Random Forest, XGBoost) to distinguish between browsers and buyers.
